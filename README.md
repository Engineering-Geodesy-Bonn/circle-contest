# circle-contest

Python implementation of the total station circle contest.

## Getting started

### Dependencies
- pyserial
- keyboard
- matplotlib
- pandas
- numpy

Installation via:

        python3 -m pip install <package-name>



### Running the scripts

- create database folder:

        mkdir db

- adjust com-port in contest.py

- run contest.py:

        python3 ./contest.py

- see leaderboard:

        python3 ./leaderboard
